<h1 align="center">Repositório para o Santander Bootcamp Fullstack Developer</h1>
 

Repositório criado para reunir e aplicar os conhecimentos e atividades do Santander Bootcamp na plataforma da DIO (a maior plataforma open education da América Latina).

## Sobre o programa
Em junho recebi a bolsa para participar do Santander Bootcamp 2022: um programa online e 100% através da plataforma da DIO, voltado para o ensino da programação de uma forma prática, interativa com experts e com mentorias exclusivas.

#### Qual o conteúdo do programa?
Oferece uma trilha completa, do básico ao avançado, sobre Java e Angular;
Abordando desde temas como Pensamento computacional, Git, Github, HTML5, CSS3.
